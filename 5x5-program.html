5x5 Güç Odaklı Antrenman Programı

**5x5 Sistemi Nedir?** Güç kazanımı için temel bileşik hareketlere odaklanan, haftada 3 gün yapılan ve her setin 5 tekrar olduğu, toplam 5 setten oluşan klasik ve etkili bir programdır. Bu program, yeni başlayanlardan ileri seviyeye kadar herkesin kuvvetini artırmak için idealdir.

1. Gün: A Antrenmanı

  • **Squat (Çömelme):** 5 set x 5 tekrar
  • **Bench Press (Yatay İtiş):** 5 set x 5 tekrar
  • **Barbell Row (Kürek Çekme):** 5 set x 5 tekrar
  • Yardımcı Egzersiz (Karın veya Kol): 3 set x 10 tekrar

2. Gün: B Antrenmanı

  • **Squat (Çömelme):** 5 set x 5 tekrar
  • **Overhead Press (Omuz İtiş):** 5 set x 5 tekrar
  • **Deadlift (Yer Kaldırma):** 1 set x 5 tekrar (Sadece 1 set)
  • Yardımcı Egzersiz (Karın veya Kol): 3 set x 10 tekrar

3. Gün: Tekrar A Antrenmanı

İkinci A antrenmanında, Bench Press yerine Omuz İtiş'i tekrar yapabilir veya programa göre değişen bir hareket uygulayabilirsiniz.

  • **Squat (Çömelme):** 5 set x 5 tekrar
  • **Bench Press (Yatay İtiş):** 5 set x 5 tekrar
  • **Barbell Row (Kürek Çekme):** 5 set x 5 tekrar
  • Yardımcı Egzersiz (Karın veya Kol): 3 set x 10 tekrar

**Önemli:** Her antrenman öncesi ısınmayı ve setler arası 2-3 dakika dinlenmeyi unutmayın. Her hafta ağırlık artışını hedefleyin.
